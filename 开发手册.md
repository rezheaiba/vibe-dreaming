# 巡游梦境 - 微信小程序开发手册

## 项目介绍

「巡游梦境」是一个帮助用户记录、整理和分享梦境的微信小程序，带有 AI 辅助精炼、扩充和解梦功能。

## 项目结构

```
vibe-dreaming/
├── app.js                 # 小程序入口文件
├── app.json               # 小程序配置（含底部标签栏）
├── app.wxss               # 全局样式
├── project.config.json    # 微信开发者工具配置
├── dreams.db              # SQLite 数据库
│
├── pages/                 # 页面
│   ├── index/             # 首页 - 我的梦境列表
│   ├── detail/            # 梦境详情页（左内容右操作）
│   ├── community/         # 社区 - 公开梦境列表
│   ├── checkin/           # 打卡 - 日历视图
│   ├── admin/             # 后台管理
│   └── comments/          # 评论页
│
├── server/               # 后端 FastAPI 服务
│   ├── main.py           # 主程序（包含所有 API）
│   └── requirements.txt  # Python 依赖
│
└── images/               # 底部标签栏图标（需要你自行添加）
```

### 页面布局

**首页 (index)** - 梦境列表
- 已登录：顶部添加梦境输入框 → 搜索框 → 按时间倒序列表
- 未登录：登录表单

**详情页 (detail)** - 左侧内容+右侧操作
- 左侧：显示梦境原始内容、AI 处理结果
- 右侧：编辑/删除/分享按钮，公开开关

**社区页 (community)** - 帖子列表形式
- 搜索框 + 瀑布流列表，展示公开梦境

**打卡页 (checkin)** - 日历形式
- 统计卡片（连续/累计） → 今日打卡按钮 → 日历网格

**管理页 (admin)** - 标签切换
- 用户管理/梦境管理切换，支持删除操作

## 部署说明

### 后端部署

1. 安装 Python 依赖：
```bash
cd server
pip install -r requirements.txt
```

2. 设置环境变量（启用 AI 功能需要）：
```bash
export OPENAI_API_KEY="your-api-key-here"
# 如果需要代理
export HTTP_PROXY="http://your-proxy:port"
export HTTPS_PROXY="http://your-proxy:port"
```

3. 启动服务：
```bash
python main.py
# 或者
uvicorn main:app --host 0.0.0.0 --port 8000
```

### 小程序配置

1. 修改 `app.js` 中的 `apiBase` 为你的实际服务器地址：
```javascript
apiBase: 'https://your-domain.com'  # 或者你的服务器 IP
```

2. 在微信开发者工具中打开项目，上传版本

3. 提交小程序审核发布

### 数据库

首次启动时，SQLAlchemy 会自动创建所有表结构：
- users - 用户表
- dream_records - 梦境记录表
- comments - 评论表
- checkins - 打卡记录表

## API 文档

后端启动后可访问 `http://your-server:8000/docs` 查看完整的 OpenAPI 文档。

## 配置说明

### 管理员
默认规则：第一个注册的用户（id=1）就是管理员，可以访问管理后台。

如需添加其他管理员，可以直接修改数据库，或在代码中添加管理员列表配置。

### AI 功能
- 需要配置 `OPENAI_API_KEY` 环境变量
- 如果不配置，AI 字段会留空，不影响基础功能使用
- 当前使用 `gpt-4o-mini` 模型，可以在代码中修改
